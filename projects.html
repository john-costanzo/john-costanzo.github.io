1.3 Royalty Report Format

See Xiufang's email:

Field Description Value Type Comments
UsageMonth The year and month the usage for YYYY-MM CHAR(7) Examples: “2016-06”, "2011-01"
AccountId Identifies the account   CHAR(10)  
ClientTypeCode How the user accesses the service   CHAR(1)  
PublicationCode Identifies the publication or source of the data   VARCHAR At most 8 characters
UsageCount Number of documents extracted for this Account and PublicationCode on the specified UsageMonth   VARCHAR string of digits representing decimal value
